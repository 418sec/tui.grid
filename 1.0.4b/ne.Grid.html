    ne. Grid

    new Grid(options)
    grid.js, line 7

    Grid public API

    Name Type Description
    options Object
    Properties:
    Name Type Attributes Default Description
    columnFixIndex number <optional>
    		 0

    Column index for fixed column. The columns indexed from 0 to this value will alaways be shown on the left side. setColumnFixIndex can be used for setting this value dynamically.
    selectType string <optional>
    		 ''

    Type of buttons shown next to the _number(rowKey) column. The string value 'checkbox' or 'radiobox' can be used. If not specified, the button column will not be shown.
    autoNumbering boolean <optional>
    		 true

    Specifies whether to assign a auto increasing number to each rows when rendering time.
    headerHeight number <optional>
    		 35

    The height of header area. When rows in header are multiple (merged column), this value must be the total height of rows.
    rowHeight number <optional>
    		 27

    The height of each rows.
    displayRowCount number <optional>
    		 10

    The number of rows to be shown in the table area. Total height of grid will be set based on this value.
    minimumColumnWidth number <optional>
    		 50

    Minimum width of each columns.
    useClientSort boolean <optional>
    		 true

    If set to true, sorting will be executed by client itself without server.
    singleClickEdit boolean <optional>
    		 false

    If set to true, text-convertible cell will be changed to edit-mode with a single click.
    scrollX boolean <optional>
    		 true

    Specifies whether to show horizontal scrollbar.
    scrollY boolean <optional>
    		 true

    Specifies whether to show vertical scrollbar.
    keyColumnName string <optional>
    		 null

    The name of the column to be used to identify each rows. If not specified, unique value for each rows will be created internally.
    toolbar Object <optional>

    The object for configuring toolbar UI.

    Properties:
    Name Type Attributes Default Description
    hasResizeHandler boolean <optional>
    		 true

    Specifies whether to use the resize hendler.
    hasControlPanel boolean <optional>
    		 true

    Specifies whether to use the control panel.
    hasPagination boolean <optional>
    		 true

    Specifies whether to use the pagination.
    columnModelList array

    The configuration of the grid columns.

    Properties:
    Name Type Attributes Default Description
    columnName string

    The name of the column.
    isEllipsis boolean <optional>
    		 false

    If set to true, ellipsis will be used for overflowing content.
    align string <optional>
    		 left

    Horizontal alignment of the column content. Available values are 'left', 'center', 'right'.
    className string <optional>

    The name of the class to be used for all cells of the column.
    title string <optional>

    The title of the column to be shown on the header.
    width number <optional>

    The width of the column. The unit is pixel.
    isHidden boolean <optional>

    If set to true, the column will not be shown.
    isFixedWidth boolean <optional>
    		 false

    If set to true, the width of the column will not be changed.
    defaultValue string <optional>

    The default value to be shown when the column doesn't have a value.
    formatter function <optional>

    The function that formats the value of the cell. The retrurn value of the function will be shown as the value of the cell.
    notUseHtmlEntity boolean <optional>
    		 false

    If set to true, the value of the cell will not be encoded as HTML entities.
    isIgnore boolean <optional>
    		 false

    If set to true, the value of the column will be ignored when setting up the list of modified rows.
    isSortable boolean <optional>
    		 false

    If set to true, sort button will be shown on the right side of the column header, which executes the sort action when clicked.
    editOption Array <optional>

    The object for configuring editing UI.

    Properties:
    Name Type Attributes Default Description
    type string <optional>
    		 'normal'

    The string value that specifies the type of the editing UI. Available values are 'text', 'text-password', 'text-convertible', 'select', 'radio', 'checkbox'.
    list Array <optional>

    Specifies the option list for the 'select', 'radio', 'checkbox' type. The item of the array must contain properties named 'text' and 'value'. (e.g. [{text: 'option1', value: 1}, {...}])
    changeBeforeCallback function <optional>

    The function that will be called before changing the value of the cell. If returns false, the changing will be canceled.
    changeAfterCallback function <optional>

    The function that will be called after changing the value of the cell.
    beforeText string <optional>

    Deprecated. (replaced with beforeContent)
    beforeContent string | function <optional>

    The HTML string to be shown left to the value. If it's a function, the return value will be used.
    afterText string <optional>

    Deprecated. (replaced with afterContent)
    afterContent string | function <optional>

    The HTML string to be shown right to the value. If it's a function, the return value will be used.
    converter function <optional>

    The function whose return value (HTML) represents the UI of the cell. If the return value is falsy(null|undefined|false), default UI will be shown. This option is available for the 'text', 'text-password', 'select', 'checkbox', 'radio' type.
    inputEvents Object <optional>

    The object that has an event name as a key and event handler as a value for events on input element.
    relationList Array <optional>

    Specifies relation between this and other column.

    Properties:
    Name Type Attributes Description
    columnList array <optional>

    Array of the names of target columns.
    isDisabled function <optional>

    If returns true, target columns will be disabled.
    isEditable function <optional>

    If returns true, target columns will be editable.
    optionListChange function <optional>

    The function whose return value specifies the option list for the 'select', 'radio', 'checkbox' type. The options list of target columns will be replaced with the return value of this function.
    columnMerge array

    The array that specifies the merged column. This options does not merge the cells of multiple columns into a single cell. This options only effects to the headers of the multiple columns, creates a new parent header that includes the headers of spcified columns, and sets up the hierarchy.

 var grid = new ne.Grid({

    columnFixIndex: 2,  //(default=0)
    selectType: 'checkbox', //(default='')
    autoNumbering: true, //(default=true)

    minimumColumnWidth: 50, //(default=50)

    scrollY: true, //(default:true)
    keyColumnName: 'column1', //(default:null)

            columnName: 'column1',
            isHidden: true

            columnName: 'column3',
            editOption: {
                type: 'text',

                    }
                }
            }

            //      - param {(number|string)} changeEvent.columnName - The field(column) name of the target cell
            //      - param {*} changeEvent.value - The changed value of the target cell
            //      - param {Object} changeEvent.instance - The instance of the Grid
            // - returns {boolean}
            changeBeforeCallback: function(changeEvent) {
                if (!/[0-9]+/.test(changeEvent.value)) {
                    alert('Integer only.');
                    return false;
                }
            },
            // - param {Object}  changeEvent
            //      - param {(number|string)} changeEvent.rowKey - The rowKey of the target cell
            //      - param {(number|string)} changeEvent.columnName - The field(column) name of the target
            //      - param {*} changeEvent.value - The changed value of the target cell
            //      - param {Object} changeEvent.instance - - The instance of the Grid
            // - returns {boolean}
            //
            changeAfterCallback: function(changeEvent) {}
        },
        {
            title: 'password input column',
            columnName: 'column5',
            width: 100,
            isFixedWidth: true,
            editOption: {
                type: 'text-password',
                beforeContent: 'password:'
            }
        },
        {
            title: 'text input when editing mode',
            columnName: 'column6',
            editOption: {
                type: 'text-convertible'
            },
            isIgnore: true
        },
        {
            title: 'select box',
            columnName: 'column7',
            editOption: {
                type: 'select',
                list: [
                    {text: '1', value: 1},
                    {text: '2', value: 2},
                    {text: '3', value: 3},
                    {text: '4', value: 4}
                ]
            },
            relationList: [
                {
                    columnList: ['column8', 'column9'],
                    // - param {*} value - The changed value of the target cell
                    // - param {Object} rowData - The data of the row that contains the target cell
                    // - return {boolean}
                    isDisabled: function(value, rowData) {
                        return value == 2;
                    },
                    // - param {*} value - The changed value of the target cell
                    // - param {Object} rowData - The data of the row that contains the target cell
                    // - return {boolean}
                    //
                    isEditable: function(value, rowData) {
                        return value != 3;
                    },
                    // - param {*} value - The changed value of the target cell
                    // - param {Object} rowData - The data of the row that contains the target cell
                    // - return {{text: string, value: number}[]}
                    optionListChange: function(value, rowData) {
                        if (value == 1) {
                            console.log('changev return');
                            return [
                                { text: 'option 1', value: 1},
                                { text: 'option 2', value: 2},
                                { text: 'option 3', value: 3},
                                { text: 'option 4', value: 4}
                            ];
                        }
                    }
                }
            ]
        },
        {
            title: 'checkbox',
            columnName: 'column8',
            editOption: {
                type: 'checkbox',
                list: [
                    {text: 'option 1', value: 1},
                    {text: 'option 2', value: 2},
                    {text: 'option 3', value: 3},
                    {text: 'option 4', value: 4}
                ]
            }

                    {text: 'option 2', value: 2},
                    {text: 'option 3', value: 3},
                    {text: 'option 4', value: 4}
                ]
            }
